## 👨🏻‍💻 스터디 방식

- **날짜**: 매주 금요일 9시 (+- 1시간)
- **진행 방식**:
  1. 스터디를 위해 조직/레포 생성
  2. 매주 각자 주제를 학습하고 노션에 정리
  3. 해당 주제와 관련된 면접 질문도 작성
  4. 스터디 날짜에 랜덤으로 2명 선정:
     - 1명: 발표 담당
     - 1명: 정리 담당
  5. 정리 담당자는 4명의 정리 내용을 취합해 최종본을 레포에 업로드
